# Vim IDE configuration

## Installation

```
ln -s /path/to/repo/base.vimrc ~/.vimrc
ln -s /path/to/repo/vim ~/.vim
```

Open vim and type `:PluginInstall`.

Then install Exuberant ctags or Universal ctags.
